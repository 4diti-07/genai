# Prompt Mode LLM Application

## Overview

By taking the help of openai and multiple coding videos, this project hopes to demonstrates how prompt engineering changes the style and quality of responses generated by a Large Language Model (LLM).

The application accepts user input, applies a selected prompt template, sends the prompt to an LLM through OpenRouter, and displays the generated response.

## Features

* User text input
* Multiple prompt modes
* OpenRouter API integration
* Empty input validation
* Console-based interface

## Prompt Modes

1. Teacher Mode
2. Summarize Text
3. Interview Question Generator
4. Study Plan Generator
5. Professional Rewrite

## Technologies Used

* Python
* OpenRouter API
* Requests
* Python-dotenv

## How to Run

Install dependencies:

pip install requests python-dotenv

Create a .env file:

OPENROUTER_API_KEY=your_api_key

Run the application:

python app.py

## Example

Input:
Machine Learning

Mode:
Teacher Mode

Output:
A beginner-friendly explanation of machine learning.

## Learning Outcome

This project demonstrates how changing prompts can significantly influence the style, structure, and usefulness of LLM-generated responses.
