AI STUDY ASSISTANT

Description:
A Python-based AI tool that uses an LLM API to help students learn and study.

Features:
1. Explain a Concept
2. Summarize Text
3. Generate Quiz Questions
4. Explain Like I'm 5

Technologies Used:
- Python
- OpenRouter API
- GPT-4o Mini

How to Run:
1. Install OpenAI package:
   pip install openai

2. Add your OpenRouter API key in the code.

3. Run:
   python task1.py

4. Choose a feature and enter your text.
